A plugin to customize the admin login page.

== Description ==

This plugin allows you to customize the admin login page with your own styles and logo.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/custom-login-page` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= How do I customize the login page? =

You can customize the login page by editing the `custom-login.css` file located in the `assets/css` directory.
